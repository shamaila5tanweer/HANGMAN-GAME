🎮 Hangman Puzzle Game (Python)

This is a simple and fun Hangman-style word guessing game built using Python. Instead of random words, the game gives you a clue, and you must guess the correct word one letter at a time.

🧩 How it works ? The game randomly selects a puzzle (clue + answer). You’ll see the clue and blank spaces for the word. Guess letters one by one. Each wrong guess builds the stick figure. You only get 5 chances — choose wisely!

🕹️ Features Random puzzle selection every time you play Clue-based guessing (makes it more interesting than classic hangman) Simple terminal interface Visual stickman progression Tracks guessed letters to avoid repeats

🎯 Rules Enter one letter per guess Correct guesses reveal letters in the word Wrong guesses add to the stickman You win if you guess the full word before the stickman is complete You lose if you reach 5 wrong guesses

▶️ How to Run Make sure you have Python installed

Run the script: python hangman.py
